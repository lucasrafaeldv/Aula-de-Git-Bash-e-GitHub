### 6. Git - Sistema de Controle de Versões Distribuído
O Git é uma ferramenta utilizada para controlar versões de projetos, permitindo registrar alterações nos arquivos e facilitar o trabalho em equipe.

### 6.2 Comandos Iniciais
- git init → inicia um repositório Git.
- git add → adiciona arquivos para versionamento.
- git status → mostra o estado dos arquivos do projeto.
- git config → configura informações do usuário no Git.
- git commit → salva as alterações realizadas.
- git log → exibe o histórico de commits.

### 7.2 Pull Requests

Pull Request é uma solicitação para enviar alterações de uma branch para outra, permitindo revisão do código antes da aprovação.
-----------------------------------------------------------------------------------------------------------------------------------
### 7.3 Resolução de Conflitos

Os conflitos acontecem quando duas alterações diferentes são feitas no mesmo arquivo. O Git permite identificar e resolver esses conflitos manualmente.
